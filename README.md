## Getting Started

```bash
npm install  # run once to install dependencies
npm run dev  # start the development server
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Assumptions

* This project uses [google-fonts](https://fonts.google.com/specimen/TikTok+Sans) [`public/font/tiktoksans-variable.tff`]

* The Development and deployment are only limited mobile-only view.
